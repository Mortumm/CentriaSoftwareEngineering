Reflection on the task:

Which CSS method is easiest to use?

I think it is safer, easier and clearer to manage page styling in a individual file, althought quick tests for visual effects are quicker to test on the HTML blocks themselves.

Which method is best for large projects?

Would prefer individual CSS files for anything larger than a single page site. Styling would become more than cumbersome for multipage sites very quickly.

What are the pros and cons of each?

Would differentiate the methods by how one should do things properly and what is something that you can use to test immediate effects on the fly and then transfer the styling to its _actual home_ in the .css file.
